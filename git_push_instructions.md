# Git Commands to Push

Run the following commands in your terminal:

1.  **Create a new branch:**
    ```bash
    git checkout -b feature/merged-updates
    ```

2.  **Add all changes:**
    ```bash
    git add .
    ```

3.  **Commit the changes:**
    ```bash
    git commit -m "Merged local features: QR Scan, Dashboard, Auth Fixes"
    ```

4.  **Push to GitHub:**
    ```bash
    git push origin feature/merged-updates
    ```
